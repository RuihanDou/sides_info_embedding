# sides_info_embedding
embedding with sides information

本项目解决数据挖掘问题模型如下：

拥有若干 item 组成的序列，每个 item 为一组特定 tag 的组合：


如有序列 

    item1 -- item2 -- item3 -- ...


如果特定组合

    item1 : tag_a, tag_b, tag_c

    item2 : tag_a, tag_c, tag_d, tag_f

    item3 : tag_d, tag_e

挖掘 tag 和 item 的embedding高维空间关系
